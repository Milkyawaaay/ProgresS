## О проекте

**ProgresS** — это веб-приложение, созданное для помощи спортсменам и любителям фитнеса в систематизации тренировочного процесса. Проект позволяет планировать тренировки, детально записывать результаты каждого подхода и наглядно отслеживать прогресс с помощью интерактивных графиков.

### Проблемы, которые решает проект
- Отсутствие систематизации тренировочного процесса
- Сложность отслеживания динамики роста рабочих весов
- Невозможность быстрого анализа эффективности тренировок
- Отсутствие структурированной истории тренировок

## Основные возможности

### 📝 Планирование и запись тренировок
- Создание тренировок с указанием даты и заметок
- Детальная запись каждого подхода: упражнение, вес, количество повторений
- Редактирование и удаление записей

### 💪 Управление упражнениями
- Создание персональной базы упражнений
- Категоризация по группам мышц (грудные, спинные, ноги, руки и т.д.)
- Поиск и фильтрация упражнений

### 📊 Визуализация прогресса
- Интерактивные графики изменения рабочих весов по каждому упражнению
- Графики динамики количества повторений
- Сравнение результатов по датам

### 🔄 Шаблоны тренировок
- Создание готовых шаблонов тренировок
- Быстрое начало тренировки на основе шаблона
- Редактирование и копирование шаблонов

### 👥 Ролевая система
- Разделение прав доступа для пользователей и тренеров
- Персонализированный интерфейс в зависимости от роли

## Модели данных (сущности)

### 1. Упражнение (Exercise)
Хранит информацию о доступных упражнениях.

| Поле | Тип | Описание |
|------|-----|----------|
| id | Integer | Уникальный идентификатор |
| name | String | Название упражнения |
| muscle_group | String | Группа мышц (например, "Грудь", "Спина", "Ноги") |
| description | Text | Описание техники выполнения (опционально) |
| created_at | DateTime | Дата создания записи |
| updated_at | DateTime | Дата последнего обновления |

### Пошаговая установка
## Клонируйте репозиторий

git clone https://github.com/Milkyawaaay/first.git 
cd ProgresS 

## Создайте и активируйте виртуальное окружение

# Создание виртуального окружения
python -m venv venv

# Активация на Windows
venv\Scripts\activate

# Активация на macOS/Linux
source venv/bin/activate

##Установите зависимости
pip install -r requirements.txt

##Настройте переменные окружения

# Django settings
SECRET_KEY=your-secret-key-here
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1

# Database settings (для PostgreSQL)
DB_NAME=workout_db
DB_USER=postgres
DB_PASSWORD=your-password
DB_HOST=localhost
DB_PORT=5432

# Для SQLite можно использовать:
# DATABASE_URL=sqlite:///db.sqlite3

##Выполните миграции базы данных

python manage.py migrate

##Создайте суперпользователя (для доступа в админ-панель)

python manage.py createsuperuser

##Загрузите начальные данные (опционально)

python manage.py loaddata fixtures/exercises.json

##Запустите сервер разработки

python manage.py runserver
